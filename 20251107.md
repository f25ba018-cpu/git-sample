###　エラーになる場合
```
sample.add(4);
print(sample.add(10)); //メソッド」を呼び出しているだけなのでエラー

// ▼エラー解消
sample.add(4);
print(sample..add(10)..add(10)); //カスケードキッドをいれることでエラー解消

// この方法で続けて計算することもできる→
print(sample..add(10)..add(10).num); //numの値を出力することもできる 

```
## NUll許容、非null許容
1. **？**: NUll許容（値にnullが代入されるのを許可）
2. **!**: nullではないことを保証する
3. **??**: nullが代入された場合の処理を定義するのに使用
```
void main() {
  print(test(10));
  print(test(null));
}
int test(int? a){
  return a ?? 0; //a!の場合は絶対nullはきませんという意味
 // nullだったら??の後が出力
}
```
